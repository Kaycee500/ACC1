Microsoft Excel Cheat Sheet for Windows

Essential Ribbon Locations

  • Home Tab: Bold, formatting, borders, number formats
  • Insert Tab: Charts, tables, pictures
  • Data Tab: Sort, Filter, Text to Columns
  • File Tab: Save, Open, Print, New

Common Shortcuts

  • Ctrl + S: Save
  • Ctrl + C: Copy
  • Ctrl + V: Paste
  • Ctrl + Z: Undo
  • Arrow Keys: Move between cells
  • Enter: Move down one cell
  • Tab: Move right one cell

Basic Formulas

  • =SUM(A1:A5): Add numbers in range A1 to A5
  • =AVERAGE(A1:A5): Average of numbers in range
  • =COUNT(A1:A5): Count cells with numbers

Saving Your Work

  1. Press Ctrl + S or click File tab
  2. Choose "Save As" for first save
  3. Pick a folder (like Documents)
  4. Type a clear file name
  5. Click "Save" button
